## :memo: Info
  - **Backend**
    - NestJS
      * API REST
      * SocketIO
      * PeerServer
      * PostgreSQL
      * Redis
      * gRPC

  - **Frontend**
    - ReactJS
      * Single Page Application

## :information_source: Getting Started

1. At the terminal, run:

```bash
$ docker-compose up -d
```

## :hammer_and_wrench: Build

```bash
$ docker-compose -f docker-compose.prod.yml build
```
